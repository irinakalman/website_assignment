A brief description of HTML

What is HTML?

HTML defines the structure of the page. It is a markup language. Every page on the web is made of HTML. The latest version of HTML is HTML5.

A brief description of CSS

What is CSS?

CSS is responsible for the visual representation of the site. It can define the background color, the width and heigh of the paragraphs etc. it defines the appearance of the website.

A brief description of Javascript

What is Javascript?

Javascript is responsible for functionality of the website. It makes it dynamic and interactive. It can be used anywhere withing the HTML. It can be used both in back end and front end.
